# Optimal-Cost-Computation-in-Matrix-Chain-Multiplication-using-DP-Technique
Streamlit application implementing Matrix Chain Multiplication using Dynamic Programming to compute the minimum scalar multiplication cost, optimal parenthesization, and DP cost table.
## Features
Finds minimum scalar multiplications,
Displays optimal parenthesization,
Shows DP Cost Table,
Built with Python and Streamlit.

## Run
pip install -r requirements.txt

streamlit run app.py
